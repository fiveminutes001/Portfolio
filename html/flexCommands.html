CSS Flex properties

Horizontal alignment

To align items to the left side of the container 

justify-content: flex-start;
#content {
display: flex;
justify-content: flex-start;
}

To align items to the right side of the container

justify-content: flex-end;
#content {
display: flex;
justify-content: flex-end;
}

To align items to the center of the container

justify-content: center;
#content {
display: flex;
justify-content: center;
}

To display items with equal spacing between them

(The outmost 2 will be on the far left and the far right)

justify-content: space-between;
#content {
display: flex;
justify-content: space-between;
}

To display items with equal spacing around them

justify-content: space-around;
#content {
display: flex;
justify-content: space-around;
}

Vertical alignement

To align items to the top of the container

align-items: flex-start;
#content {
display: flex;
align-items: flex-start;
}

To align items to the bottom of the container

align-items: flex-end;
#content {
display: flex;
align-items: flex-end;
}

To align items to the vertical center of the container

align-items: center;
#content {
display: flex;
align-items: center;
}

To align items to the baseline of the container

align-items: baseline;
#content {
display: flex;
align-items: baseline;
}

To strech items to fit the container

align-items: stretch;
#content {
display: flex;
align-items: stretch;
}

Horizontal order

To put items in left-to-right order

flex-direction: row;
#content {
display: flex;
flex-direction: row;
}

To put items in right-to-left order

flex-direction: row-reverse;
#content {
display: flex;
flex-direction: row-reverse;

//Notice that when you set the direction to a reversed row or column, start and end are also reversed. 

}

Vertical order

To put items in top-to-bottom order

flex-direction: column;
#content {
display: flex;
flex-direction: column;

//Notice that when the flex direction is a column, justify-content changes to the vertical and align-items to the horizontal. 

}

To put items in bottom-to-top order

flex-direction: column-reverse;
#content {
display: flex;
flex-direction: column-reverse;

//Notice that when you set the direction to a reversed row or column, start and end are also reversed. 

}

Internal order

To put items in bottom-to-top order and change their internal order 

flex-direction: column-reverse;
#content {
display: flex;
}

//Sometimes reversing the row or column order of a container is not enough. In these cases, we can apply the order property to individual items. Let's say we have 3 items. By default, items have a value of 0, but we can use this property to also set it to a positive or negative integer value (-2, -1, 0, 1, 2). It's like z-index, and can be applied to several elements within the same class.

.yellow {
order:-2;
}

Across lines

To put every item in a single line

flex-wrap: nowrap;
#content {
display: flex;
flex-wrap: nowrap;
}

To wrap items to additional lines

flex-wrap: wrap;
#content {
display: flex;
flex-wrap: wrap;
}

To wrap items to additional lines in reverse

flex-wrap: wrap-reverse;
#content {
display: flex;
flex-wrap: wrap-reverse;
}

Across lines and Vertical design

To put every item in a single line and do also vertical design 

flex-flow: column wrap;
#content {
display: flex;
flex-flow: column wrap;

//The two properties flex-direction and flex-wrap are used so often together that the shorthand property flex-flow was created to combine them. This shorthand property accepts the value of the two properties separated by a space. For example, you can use flex-flow: row wrap to set rows and wrap them.

}

To put every item in a single line and do also vertical design 

flex-flow: row nowrap;
#content {
display: flex;
flex-flow: row nowrap;

//The two properties flex-direction and flex-wrap are used so often together that the shorthand property flex-flow was created to combine them. This shorthand property accepts the value of the two properties separated by a space. For example, you can use flex-flow: row wrap to set rows and wrap them.

}

Individual design

To style an individual item

align-self: flex-end;
#content {
display: flex;
align-items: flex-start;
}

//Another property you can apply to individual items is align-self. This property accepts the same values as align-items and its value for the specific item.

.yellow {
align-self: flex-end;
}

Lines space

To pack lines at the top of the container

align-content:flex-start;
#content {
display: flex;
flex-wrap: wrap;
align-content:flex-start;

//This can be confusing, but align-content determines the spacing between lines, while align-items determines how the items as a whole are aligned within the container. When there is only one line, align-content has no effect.

}
